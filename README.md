# Arino
Arduino OTA upload with linkit smart 7688 or raspberry pi

## Install on micropc
### Linkit Smart 7688
Login ssh to your linkit smart 7688 then enter bottom command
```sh
# wget http://www.file.host-1gb.com/arino/install.sh
# sh install.sh
```
